Clone the repository:

```bash
git clone https://github.com/Inzila2130/HousePriceAI.git
```

Go to the project folder:

```bash
cd HousePriceAI
```

Create a virtual environment:

```bash
python -m venv venv
```

### Windows


