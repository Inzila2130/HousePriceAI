## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/Inzila2130/HousePriceAI.git
```

Go to the project folder:

```bash
cd HousePriceAI
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
