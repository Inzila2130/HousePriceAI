# 🏠 HousePriceAI

An AI-powered web application that predicts residential house prices using a Machine Learning Linear Regression model. Built with Flask, Scikit-Learn, HTML, CSS, and JavaScript.

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/HousePriceAI.git
```

Go to the project folder:

```bash
cd HousePriceAI
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
