Department 05

About HousePriceAI

HousePriceAI is a Machine Learning based web application developed to estimate residential property prices using Artificial Intelligence.

Project Overview

This project demonstrates the complete Machine Learning workflow, from data preprocessing and feature engineering to model training, deployment with Flask, and an interactive web interface.

Users can enter property details, receive AI-powered price predictions, explore property snapshots, and understand the model through interactive analytics.

Technology Stack

Python Flask Scikit-Learn Pandas NumPy HTML5 CSS3 JavaScript

Developer

Inzila Danish Khan

HousePriceAI was developed as a Machine Learning project to demonstrate AI-powered property price prediction through an interactive virtual headquarters experience.
